## 斜体、粗体、删除线、分隔线

*斜体* 或 _斜体_

**粗体** 或 __粗体__

***粗斜体*** 或 ___粗斜体___

~~删除线~~

分隔线：一行中有三个以上相同的 `*` 或 `-` 或 `_`，除了空格可任意添加，不能有其他字符
***

## 引用

可添加任意个 `>` 表示引用层级
> 引用
> > 引用

## 无序列表

行首加上 `*` 或 `+` 或 `-`，若行首再加 2 个空格表示二级列表，以此类推

- A
- B
  - A
  - B

## 有序列表

行首加上 `数字.`，若行首再加 3 个空格表示二级列表，以此类推

1. A
2. B
   1. A
   2. B

## 代码块

`行内代码块`

多行代码块：

```html
<h1>123</h1>
```

```css
h1 {
    font-size: 24px;
}
```

```js
const add = (a, b) => a + b

add(1, 2)
```

也可使用每行行首加 4 个空格或 1 个 tab 表示多行代码块

## 表格

以 `|` 分隔单元格，以 `-` 分隔表头（默认左对齐），`:-` 左对齐，`:-:` 居中，`-:` 右对齐

|  表头  |  表头  |  表头  |
|   -    |  :-:   |   -:   |
|   左   |   中   |   右   |
| 单元格 | 单元格 | 单元格 |
